PySam
#####


PySam provide a Python scripting interface for SysML2 models.
You can load models from any SysML2 tools who provide an implementation of the standard API.
You can also manipulate models, like parse, or edit them, and push the modification on your modeling tool.

This library intetend to work first with SAM, the SysML2 modeling tool of Ansys
