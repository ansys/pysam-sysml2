Download a single diagram and save it to disk.

diagram_idstr

The diagram id to download.

pathstr or Path

Destination file or directory.

file_formatstr

Format of the diagram (e.g., ‘svg’, ‘png’, ‘jpeg’).

filenamestr, optional

Name of the file. Default is diagram ID.
